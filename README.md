# 🏆 Portal de Esportes

Um site informativo e visualmente atrativo com foco em notícias esportivas e diversas modalidades como **futebol**, **basquete**, **vôlei**, **futebol americano** e **artes marciais**.

> ⚠️ **Aviso:** Todas as notícias exibidas neste portal são **fictícias**, criadas apenas para fins **educacionais e demonstrativos**.

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar e aplicar conhecimentos em **HTML**, **CSS** e **JavaScript**, criando uma experiência web moderna, responsiva e acessível.

---

## ✨ Funcionalidades

- ✅ Apresentação de notícias esportivas fictícias
- ✅ Seções organizadas por modalidade
- ✅ Galeria de imagens para cada esporte
- ✅ Formulário de contato funcional
- ✅ Layout responsivo e estilizado

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)
- Font Awesome (ícones)

## 🚀 Como Executar o Projeto

1. Clone o repositório:

```bash

```

2. Acesse a pasta do projeto:


3. Abra o arquivo `index.html` no seu navegador.
